Help

What can we help you?
  • Need to reload product listing or case needed report?
  • Never get an output?
  • Schedule not work?

Upload and Submit

  • Use the "Choose File" button to reupload the files and click "Submit" can reupload files.
  • You need to upload both files at the same time.
  • Uploaded file will not be overwitten when next time submit button is successfully clicked.

Output

  • If no output is provided, it may be due to following reasons:
    • The order of files uploaded is wrong.
    • The connection of the frontend and the backend is broken or failed.
    • New information that can not be handled automatically is added to product listing or case needed report.
    • The number of batches included is so large that the algorithm can not handle at this time.
  • If output is incorrect or infeasible, it may be due to following reasons:
    • The order of files uploaded is wrong.
    • New information that can not be handled correctly is added to product listing or case needed report.
    • The number of batches included is so large that the algorithm need more time to provide a better schedule.

Schedule Limitaion

  • Our program is based on a free open-source solver. Therefore the efficiency of solving is low. Paid solvers are proved to have much better performance with the same model.
  • If your factory adds more equipment and therefore generates more constraints, our program can not automatically handle them.
  • Our program requires an amount of time to run. Therefore, you may need to wait some time before seeing a schedule outputed.
